# Published Blog

My blogs to share with people.

## Getting Started

### Prerequisites

``` bash
# jekyll
gem install bundler jekyll
```

## Deployment

``` bash
bundle exec jekyll serve
```

* [Jekyll](https://jekyllrb.com/) - The web framework used
